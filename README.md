# Shader编辑器
编辑器目前处于立项阶段
预计于2025年12月30号前发V1.0.0测试版 /c
2025年12月7日 ShaderEditor编辑器 V0.0.1 测试版发布 /c
1.目前版本的主窗口大部分UI的触发事件暂未实装 /c
2.打开主窗口后点击左上角启动即可进入编辑器选择界面 /c
3.目前只有常量编辑器实装且经过测试，其他编辑器处于开发阶段 /c
4.常量编辑器使用方法为： /c
	1.打开编辑器后打开 钢铁雄心4 的 constants.fxh 文件，路径：Hearts of Iron IV\gfx\FX\constants.fxh /c
	2.成功载入 constants.fxh 文件后,请载入与常量文件同名的 .expl文件(解释文件) /c
	3.编辑器提供两种保存模式 /c
		1.直接保存到原文件 /c
		2.保存编辑后的文件到另一位置 /c
5.编辑器使用.NET框架开发，使用前请安装 .NET 8.0 (运行时)   下载地址: https://builds.dotnet.microsoft.com/dotnet/Sdk/8.0.416/dotnet-sdk-8.0.416-win-x64.exe
